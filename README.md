# Q-share
## Introduction
`q-share` allows you to share files, links, images and more between mobiles and computers easliy using qrcodes.
## Installation
To install `q-share` from github page:
```properties
git clone https://www.github.com/bashar-mithan/q-share
```
After clonning the repestory, you build it. It can be biuld using python.
```properties
python3 main.py
```
